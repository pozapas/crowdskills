## Review Steps

1. Disable active recognition and tracking.
2. Use `show only people` for the current ID.
3. Jump to trajectory start, middle, and end.
4. Inspect the full path with an appropriate frame window.
5. Check correction-tab candidates for equality, velocity, length, and inside tests.
6. Save `.trc` after correction blocks.
7. Export analysis format only after the checklist is complete.
